# 🎨 Rezaan Achmat Portfolio Website

A modern, responsive personal portfolio website showcasing projects and skills. Built with cutting-edge web technologies for optimal performance and user experience.

[![Deploy to Pages](https://github.com/Rezaan91/Rezaan91-My-Portfolio/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/Rezaan91/Rezaan91-My-Portfolio/actions/workflows/deploy-pages.yml)

## ✨ Features

- **Responsive Design** - Beautiful layouts that work on all devices
- **Fast Performance** - Optimized with Vite for lightning-fast load times
- **Modern Styling** - Professional UI components with Tailwind CSS and shadcn/ui
- **Type-Safe** - Built with TypeScript for reliability and better developer experience
- **Auto-Deployment** - GitHub Pages integration for continuous deployment

## 🚀 Quick Start

### Prerequisites

- Node.js 22+ and npm

### Development

1. **Install dependencies:**

```sh
npm install
```

2. **Start the development server:**

```sh
npm run dev
```

The site will be available at `http://localhost:5173`

3. **Build for production:**

```sh
npm run build
```

4. **Preview the production build:**

```sh
npm run preview
```

## 🌐 Deployment

This repository is configured for automatic deployment to GitHub Pages.

**How it works:**

- Push to `main` branch
- GitHub Actions automatically builds and deploys your site
- Your portfolio is live at: https://rezaan91.github.io/Rezaan91-My-Portfolio/

The deployment workflow is configured in `.github/workflows/deploy-pages.yml`
